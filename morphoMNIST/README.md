## Environment Setup
### Python Virtual Environment
These codes share the same setup as the main code.
### MorphoMNIST API
Download the `morphomnist` directory from the MorphoMNIST GitHub repository: [Morpho-MNIST](https://github.com/dccastro/Morpho-MNIST). 
Then, copy it to `morphoMNIST/`.
As results, the API files are in `morphoMNIST/morphomnist/`.

## Data Download

To download the required dataset, follow the steps below:
1. Open the MorphoMNIST GitHub repository: [Morpho-MNIST](https://github.com/dccastro/Morpho-MNIST)
2. Download the *global* dataset, which contains digit and thickness labels.
3. Unzip the file and copy it to the following path: `morphoMNIST/`

## Execution

You can reproduce the experiment using the `test.ipynb` file.
